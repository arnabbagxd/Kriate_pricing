# Kriate Studio — Service Offerings

A professional website showcasing all Kriate Studio services and pricing tiers.

## Project Structure

```
dev-kriate/
├── index.html           # Main HTML file
├── css/
│   └── styles.css       # All CSS styles
├── js/
│   └── script.js        # JavaScript functionality
├── assets/              # Future folder for images, fonts, etc.
├── .gitignore          # Git ignore file
└── README.md           # This file
```

## How It Works

### Pages
- **01 — Brand Identity** - Complete brand system services
- **02 — Design Retainer** - Monthly design & content services
- **03 — Performance Marketing** - Meta + Google ad campaigns
- **04 — Website & SEO** - Website development and SEO services

### File Breakdown

**index.html**
- Main entry point
- Contains all the HTML structure
- Links to external CSS and JavaScript

**css/styles.css**
- All styling for the project
- Color variables defined in `:root`
- Responsive design with media queries

**js/script.js**
- Page navigation functionality
- Handles switching between different service pages

## To Use This Project

1. Open `index.html` in your web browser
2. Click the navigation buttons at the top to switch between services
3. Each service shows different pricing tiers and features

## What Each Folder Does

| Folder | Purpose |
|--------|---------|
| `css/` | Stores all stylesheet files |
| `js/` | Stores all JavaScript files |
| `assets/` | For images, fonts, and other media files (empty for now) |
